# Bakery-Website

A small somewhat messy yet highly featured website for a bakery in the local area. The website was designed as part of my Alevel and Btech computing course.

## Features

- Admin and Customer Views
- Ordering system aswell viewing placed orders
- Login and register systems
- Embedded videos and links
- Checkout systems

## Built With

- HTML
- CSS
- Flask
- Python
- Functionality from SQLite databases

## What i learned

- Styling websites with HTML
- Interacting with databases through a website
- Layout and Structure with HTML
- Using Python's Flask module to host small projects locally
- Submitting images onto a website
